- TOP_RATED: `https://api.themoviedb.org/3/movie/top_rated`

- DETAIL: `https://api.themoviedb.org/3/movie/{movie_id}`
- POSTER: `https://api.themoviedb.org/3/movie/{movie_id}/images`
